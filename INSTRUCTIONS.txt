# ── Troubleshooting ───────────────────────────────────────────────────────────
#
# "ModuleNotFoundError": make sure (.venv) is active and you ran pip install.
#
# task1 finishes but corpus is tiny:
#   The IITJ website may have blocked the crawler. Try reducing DELAY_SEC
#   or add a manual wait. The PDF text alone is still valid for the assignment.
#
# task2 scratch training is very slow:
#   Normal — pure NumPy on a large corpus takes time. You can reduce MAX_PAGES
#   in task1_corpus.py (e.g. to 100) or reduce epochs in task2_train.py (e.g. to 1).
#
# task4 t-SNE raises "perplexity must be < n_samples":
#   This means too few words from WORD_GROUPS were found in the model vocab.
#   Add more words to WORD_GROUPS in task4_visualize.py or use PCA only.
#
# torch not found in Problem 2:
#   Make sure you installed requirements.txt inside the venv, not globally.
